## 通用维度

- **风格**: 正式 (conf=0.90, 可逆) → skip
  `"借：管理费用-办公费 10,000  贷：银行存款 10,000"`
- **长度**: 中/长 (conf=0.50, 可逆) → display
- **格式**: 表格+文字 (conf=0.80, 可逆) → skip
- **深度**: 深度 (conf=0.50, 可逆) → display
- **受众**: 财务/税务专业人员 (conf=0.90, 不可逆) → display
- **语言**: 中文 (conf=0.90, 可逆) → skip

## 特色选项

### 会计准则 (conf=0.60, 不可逆) → display
- **企业会计准则** (default) — 上市公司/大中型企业
- 小企业会计准则
- IFRS 国际财务报告准则

### 分录详细程度 (conf=0.60, 可逆)
- **一级+明细科目** (default)
  `"借：管理费用-办公费-耗材 10,000  贷：银行存款-建行XX支行 10,000"`
- 仅一级科目
  `"借：管理费用  贷：银行存款"`
- 含辅助核算（部门/项目/客户/供应商）

### 纳税人身份 (conf=0.50, 不可逆) force
- **一般纳税人** (增值税 13%/9%/6%)
- 小规模纳税人 (征收率 3%/5%)

### 凭证格式 (conf=0.80, 可逆) → skip
- **标准记账凭证** (default) — 摘要+借方+贷方+金额+日期+凭证号
- 含附件清单+审核栏+制单人

## 盲区预警

- 税务政策时效: 税率和优惠政策可能更新，标注引用政策截止日期
- 地区差异: 部分地区特殊优惠政策，必要时确认
- 本 Skill 提供财税参考，以专业会计师/税务师意见为准
