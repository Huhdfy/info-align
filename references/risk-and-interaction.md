# 风险矩阵与交互指南

阶段2（生成预填模板）时加载。包含 Agent 内部决策规则和用户输出格式规范。

---

## 一、选项风险分类矩阵（Agent 内部）

| confidence | reversible | 策略 | 用户侧表现 |
|---|---|---|---|
| ≥ 0.60 | 可逆 | 直接默认 | **不展示**该字段 |
| ≥ 0.60 | 不可逆 | 预填，请扫一眼 | 展示在"默认配置"区，无特殊标记 |
| < 0.60 | 可逆 | 预填，建议确认 | 展示在"默认配置"区，标 ⚠️ |
| < 0.60 | 不可逆 | 强制用户提供 | 展示在"需要你确认"区，标 ❓，不设默认值 |

Agent 内部使用 confidence 和 strategy 做决策，对外输出**只显示决策结果**，不输出数值或策略名。

---

## 二、可逆性判断规则（Agent 内部）

### 可逆
风格、长度、格式、语调、图表要求、数据颗粒度、RSVP、附件、CTA、数据支撑

### 不可逆
收件人、发布日期、数据来源路径、会计准则、纳税人身份、适用法律体系、绩效框架、目标服务器地址、API密钥、产品类型、引用格式、发布平台

## 三、用户输出格式规范（强制）

### 预填模板

按以下结构输出，只用三色标记 ✅ ⚠️ ❓。禁止出现：confidence 数值、"可逆/不可逆"、"skip/display/display_warning/force"、HTML 注释、文件路径引用。

```
关于这个任务，我预设了以下配置。你只需回复要改的部分。

已确认：
  ✅ [字段]：[值]

默认配置：
  ⚠️ [字段]：[值]
     [1-2句自然语言描述，展示该选项下的输出风格，不标注"示例："前缀]

需要你确认：
  ❓ [字段]：[提示需要提供什么信息]

可能需要但你暂时不必确认：
  ⚠️ [字段]：[已使用的默认值]
     [简要说明潜在风险]

---
回复"确认"直接开始，或告诉我哪项需要调整。
```

### 板块规则

- **已确认**：用户明确提供的值。无则不展示此板块。
- **默认配置**：Agent 预设的值（包括低 conf 的 ⚠️ 项）。无则不展示此板块。
- **需要你确认**：conf<0.60 且不可逆的字段，必须用户提供。无则不展示此板块。
- **可能需要但你暂时不必确认**：盲区预警（前置条件已设默认值可跳过，但提醒风险）。无则不展示此板块。

### 确认单

```
【需求确认单】

任务：[一句话描述]
类别：[12大类名称]

已确认：
  ✅ [字段]：[值]

默认配置：
  ⚠️ [字段]：[值]

需要你补充：
  ❓ [字段]：[说明]

---
回复"确认"开始执行，或回复要修改的项。
```

板块规则与预填模板相同。确认单是预填模板的"确认后快照"，去掉了引导语和 Reason 句子。

### Validate 模板

用户回复"确认"后，回显一句话理解摘要，不重新输出整个确认单：

```
好的，我确认一下：你需要的是一份[任务]，按[格式]输出，风格[风格]，重点覆盖[关键维度]。没问题我现在就开始。
```

用户回复"开始"/"没问题"/"ok"之后，交给下游 Skill。

### 示例规则

示例片段用于让用户直观感受选项效果。规则：
- 不标注"示例："前缀。直接以自然语言或引用块呈现
- 风格类选项：给出 1-2 句开头示例，展示语调
- 格式类选项：给出结构描述（如"按行业空间→商业模式→财务→估值组织"）
- 操作类选项：给出行为描述（如"将保留标题层级和表格，不保留字体颜色"）

### ❓ 和 ⚠️ 的语义

- ✅ = 你已经告诉我了，已确认
- ⚠️ = 我按默认值设了，可以直接用，但你可以改
- ❓ = 这个需要你告诉我，否则没法继续

## 四、自然语言修改解析（Agent 内部）

优先级：
1. 关键词精确匹配 — "风格轻松" → 维度="风格", 值="轻松"
2. 模糊匹配 — "别太正式" → 维度="风格", 值="轻松/中性"
3. 否定式 — "不要太长" → 维度="长度", 取默认值范围下限
4. 增项式 — "加一列估值数据" → 维度="图表要求", 值改为"嵌入表格"
5. 无法匹配 → 追问一句

容错：
- "正式" → 优先匹配风格
- "加上XXX" → 先查特色选项能否承载
- "不要XXX" → 检查是否当前默认值；若是→改备选值；不是→确认意图

---

## 五、最小交互原则（Agent 内部）

- conf≥0.60 且可逆：不展示
- 下游 Skill 声明为"内部使用"：不展示
- 示例中已隐含的选项：不单独列出

修改范围收敛：
- "改风格为轻松" → 只改风格
- "全按默认" → 全部采用默认值

---

## 六、响应处理（Agent 内部）

1. "确认" → 锁定所有字段 → 生成确认单
2. "改A为B" → 解析 → 只修改指定字段 → 输出简化确认单（仅变更项和❓项）
3. "我也不确定XX" → 触发象限❸维度展开

---

## 七、自适应调整因子（平台层，Agent 内部）

```
confirm: +0.10 | overridden: -0.20 | max: 0.95 | min: 0.10
```

非对称、连续3次同向减半步长、conf=0.95后锁定（漂移检测重置）。
