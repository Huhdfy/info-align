## 通用维度（精简）

- **格式**: API生成结果 (conf=0.90, 可逆) → skip
- **语言**: 中文 (conf=0.50, 可逆) → display_warning

## 特色选项

### 输出比例（图片/视频生成） (conf=0.60, 可逆)
- **16:9 横版** (default)
- 9:16 竖版
- 1:1 正方形
- 4:3 传统
- 自适应

### 分辨率/质量 (conf=0.60, 可逆) → skip
- **标准 2K** (default)
- 高清 4K
- 快速预览（低分辨率）

## 盲区预警

- 生成结果受模型能力限制，复杂场景建议拆分
- API可用性和速率限制可能影响速度
