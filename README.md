## 帮助用户与AI信息对齐的SKILL
我们认为，规范化和可重复的一件事，它就可以被固定成一个SKILL。
现在的各种效率提高工具层出不穷，AI和人之间的交流将会变成一个影响效率的不可忽视的环节。
而这个SKILL的目的就是将AI与用户交流这件事本身给规范化，成为一个固定、标准的流程。
